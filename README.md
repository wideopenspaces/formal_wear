# FormalWear

FormalWear helps you create Form Objects with required and optional attributes and gets all fancy widdit.

## Installation

Add this line to your application's Gemfile:

    gem 'formal_wear'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install formal_wear
